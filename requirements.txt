1. Project Objective
The purpose of this project is to train you as a 
Full Stack Web Developer
 by developing a production-ready E-Commerce application from scratch.
Unlike tutorial-based projects, this assignment follows a real software company’s development workflow. You are expected to analyze requirements, design the database, build secure backend APIs, test them thoroughly, integrate them into a modern Next.js frontend, and deliver a complete, scalable application.
You are responsible for the 
entire software development lifecycle (SDLC)
 including planning, development, testing, debugging, documentation, and deployment.

37. Professional Expectations
As an intern, you are expected to work as a professional software developer.
You must:
Plan your work before writing code.
Build both the backend and frontend independently.
Test every API in Postman before integrating it into the frontend.
Write clean, modular, and reusable code.
Follow REST API standards and proper project architecture.
Keep frontend and backend code organized.
Commit your work to Git regularly with meaningful commit messages.
Document your APIs, database, and setup instructions.
Ask questions whenever requirements are unclear instead of making assumptions.
Test every feature before marking it as complete.
Fix bugs promptly and maintain code quality throughout the project.
Deliver production-quality work that is secure, scalable, maintainable, and well documented.
This assignment is intended to simulate the expectations of a real software company. Treat every task as if you are developing software for a paying client.
